# KTravel Website

A modern travel website for selling Korean tour packages, built with Next.js and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - Reusable React components
  - `Hero.tsx` - Hero section component
  - `TourCard.tsx` - Individual tour card component
  - `TourCardList.tsx` - List of tour cards with sample data

## Features

- Hero section with call-to-action
- Tour product cards with sample data
- Responsive design with Tailwind CSS
- TypeScript for type safety
